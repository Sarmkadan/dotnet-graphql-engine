# Contributing

Welcome to `dotnet-graphql-engine`!

## Getting Started

1. **Fork** the repo.
2. **Clone**: `git clone https://github.com/sarmkadan/dotnet-graphql-engine.git`
3. **Build**: `dotnet build`
4. **Test**: `dotnet test`

## Workflow

1. Create a branch: `feature/name` or `fix/name`.
2. Make your changes, following `[*.editorconfig]` rules.
3. **Add tests** for new functionality.
4. Run `dotnet test`.
5. Open a Pull Request targeting `main`.

## Guidelines

- Keep PRs focused.
- Add/update XML documentation for public APIs.
- Run `dotnet format`.
- Ensure all CI checks pass.
